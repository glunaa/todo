# ✅ React + TypeScript Todo App

A simple and elegant Todo application built using React and TypeScript. This project demonstrates component-based architecture, state management, and type safety in a modern frontend stack.

## 🧰 Tech Stack

• React – UI library for building interactive interfaces
• TypeScript – Strongly typed language for safer and scalable code
• CRA – Fast development environment (customize based on your setup)
• CSS Modules / Boostral CSS – Styling (adjust based on your preference)


## ✨ Features

• Add, edit, and delete todos
• Mark tasks as complete/incomplete
• Responsive design for mobile and desktop
